# Host Migration Guard

A Python project for monitoring and observability of deployments.

## Features

* Real-time monitoring of deployment status and health indicators
* Alerting system for failures or anomalies
* Metrics collection for CPU, memory, and network usage
* Integration points for external observability platforms

## Usage

1. Install the project using `poetry install`
2. Run the tests using `pytest`
3. Use the `HostMigrationGuard` class to monitor and observe your deployments
